# Mayura Manawadu — Portfolio

Personal academic website of Mayura Manawadu, PhD Candidate at the University of Melbourne — Mixed Reality, Computer Vision, Novel View Synthesis.

Live: https://mayura1996.github.io/Mayura/

## Structure

```
.
├── index.html              Main portfolio page
├── stylesheet.css          Global styles
├── script.js               Interactions (typing, scroll reveal, background, counters)
├── assets/
│   └── cv.pdf              Curriculum vitae
├── images/                 Photos, project videos, favicon
├── blog/                   Travel blog posts
└── projects/               Individual project pages
    ├── abhises/            AR Tour Guide (Sigiriya)
    ├── 6dof/               6DoF Pose & Focal Length Estimation
    ├── traffic-sign/       Voice-assisted Traffic Sign Recognition
    ├── accident-prediction/  US Accident Prediction
    └── cricket-game/       C++ Console Cricket Game
```

## Development

It's a static site — no build step. Open `index.html` in a browser or serve the folder over any static HTTP server.
